4. Tag de Tabela :
Qual tag define uma linha em uma tabela HTML?

A tag < tr > faz as linhas da tabela Preço Quantidade
A tag < th > é colocada dentro da tag tr para fazer o cabeçario da tabela R$ 50,00 10
A tag < td > é colocada dentro da tag tr para colocar as celulas de dados na tabela R$ 100,00 5
